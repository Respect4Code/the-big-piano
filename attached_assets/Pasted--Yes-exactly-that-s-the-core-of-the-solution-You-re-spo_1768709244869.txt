.modal-overlay {

  padding: 24px;
  width: 90%;
  max-width: 360px;
  max-height: 80vh;            /* ← Key: limits height to ~80% of viewport */
  overflow-y: auto;            /* ← Enables vertical scrolling */
  -webkit-overflow-scrolling: touch;  /* ← Smooth iOS momentum */
  box-shadow: 0 10px 25px rgba(0,0,0,0.4);
}

/* Body scroll lock when modal is open */
body.modal-open {
  overflow: hidden;
  height: 100vh;
}

JavaScript (simple toggle for body lock):js
// When opening modal
function openModal() {
  document.body.classList.add('modal-open');
  // Optional: save scroll position if needed
  const scrollY = window.scrollY;
  document.body.style.position = 'fixed';
  document.body.style.top = `-${scrollY}px`;
}

// When closing modal
function closeModal() {
  document.body.classList.remove('modal-open');
  // Restore scroll
  const scrollY = -parseInt(document.body.style.top || '0', 10);
  document.body.style.position = '';
  document.body.style.top = '';
  window.scrollTo(0, scrollY);
}

Why This Worksmax-height: 80vh ensures the modal never gets taller than the screen, even with keyboard open.
overflow-y: auto + -webkit-overflow-scrolling: touch makes the content scroll smoothly inside the modal.
Body lock prevents double-scroll jank (background page moving underneath).
Tested pattern — used in most modern web apps (banking, auth modals, etc.).
